# django-crud-example
A simple crud in django

Used:

- Django 4
- Bootstrap 4
- Postgres

Get Started:

```
#in your env

$ sudo pip install -r requirements.txt

$ ./manage.py migrate

$ ./manage.py runserver

#django admin

$ python manage.py createsuperuser

access /admin

```
